# ARM---BASED-REAL-TIME-DIGITAL-AUTOMOTIVE-INSTRUMENT-CLUSTER-WITH-CAN-PROTCOL-INTEGRATION-
Developed a real-time automotive instrument cluster using LPC2129 with CAN protocol. Enabled reliable node communication via CAN transceiver, reducing wiring complexity, cost, and weight. Displays live vehicle data (speed, indicators, headlights) with a scalable, modular design.
